# Topic 15: File I/O

## Context
Go I/O - interface-based, simpler than C++ streams.

## Structure
- `basic/` - Reading/writing files, os package
- `intermediate/` - bufio, io.Reader/Writer, JSON

## Key Focus
- os.Open, os.Create
- io.Reader/Writer interfaces
- bufio for buffering
- JSON encoding/decoding
